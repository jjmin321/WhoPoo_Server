# WhoPoo_Server

WHOPOO는 리그오브레전드 API를 사용하여 데이터를 정리하고 보여주는 서비스입니다.

## Stack
|                      | Android     | Server        | 
|:--------------------:|:---------------:|:------------------:|
| Developer | 남규석 | 제정민 | 
| Develop Language | Kotlin | GO| 
| Develop Tool     | Android Studio  | Visual Studio Code | 

## Motive 

- 롤 하는데 화나서 누가 못했는지 확실하게 알아보기 위해서

## API

## /matchlists
    # 유저 이름을 입력받아 최근 게임 전적을 반환해줌

    - Parameter
        - name (required)
    - Query Parameter
        - startIndex (Optional)
        - endIndex (Optional)
